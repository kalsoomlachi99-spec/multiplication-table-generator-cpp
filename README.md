*# Multiplication Table Generator (C++)*



*A simple C++ console application that generates one or multiple multiplication tables with custom limits and input validation.*



*## Features*



*- Generate multiplication tables for any number*

*- Set a custom limit for each table*

*- Create multiple tables in a single execution*

*- Input validation for user entries*

*- Clean and easy-to-read console output*

*- Function-based program design*



*## Technologies Used*



*- C++*

*- Standard Input/Output Library (`iostream`)*



*## How It Works*



*1. The user enters how many multiplication tables they want to create.*

*2. For each table:*

&#x20;  *- Enter the number.*

&#x20;  *- Enter the desired limit.*

*3. The program validates the input.*

*4. The multiplication table is displayed.*



*## Example*



*### Input*



*How many tables you want to create? 2*



*For which number do you want to create a multiplication table? 5*



*Enter limit for multiplication table: 10*



*For which number do you want to create a multiplication table? 7*



*Enter limit for multiplication table: 5*



*### Output*



*5 x 1 = 5*



*5 x 2 = 10*



*5 x 3 = 15*



*...*



*5 x 10 = 50*



*7 x 1 = 7*



*7 x 2 = 14*



*7 x 3 = 21*



*...*



*7 x 5 = 35*
